ダウンロード有難うございます。このキャラクターは電撃文庫FightingClimaxのオリジナルグラフィック
をベースにPotS/Infiniteスタイルに構成したものです。
電撃FCと違い、３ボタンから６ボタンに無理やり変更させています。
SAO主人公の黒の剣士キリトくんです。
二刀流になったり片手剣に戻したり忙しいキャラです。（本来切り札で強化するのですが……）


=====< 技表 >=====

 U - 上          x - 弱パンチ        a - 弱キック
 D - 下          y - 中パンチ        b - 中キック
 F - 前          z - 強パンチ        c - 強キック
 B - 後          p - パンチ          k - キック
 s - スタート    2p- パンチ同時押し  2k- キック同時押し
 
 (空中)   - 空中のみ
 (空中可) - 空中でも可能
 (EX)     - パンチボタン又はキックボタンを２つ同時押しすると必殺技が強化される。
 (MAX)    - パンチボタン又はキックボタンを２つ同時押しすると超必殺技が強化される。

  .回避攻撃                                ⇒ p / k
.前方回避                                 F + x + a
.後方回避                                 B + x + a
.ブロッキング (上中段)                    tap F
.ブロッキング (下段)                      tap D
.ブロッキング (空中)                      tap F                               (空中)
.パワーゲージ溜め                         hold y + b
.ガードキャンセル回避                     F/B + a + x                         (地上ガード中に) (パワーゲージ1本)
.ゼロカウンター                           F + 2p / 2k                         (地上ガード中に) (パワーゲージ1本)
.カスタムコンボ                           z + c                               (空中可) (パワーゲージ1本)
.受身                                     2p / 2k                             (落下中、または地面につく瞬間)
